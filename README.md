# BG Remover Bot

A Telegram bot that removes the background from images automatically.  
Simply send a photo or reply to an image with `/removebg` to get a transparent background image.  

Developed by: https://github.com/mahinexe

---

## Features

- Remove background from images (JPG, PNG, WEBP)  
- `/start` command – Welcome message and usage instructions  
- `/removebg` command – Remove background by replying to an image  
- `/about` command – Information about the bot and developer  
- Works locally or deployed online for 24/7 usage
